Godroll rater is an idea by FtpApoc, where rolls are given scores based on their perks and masterworks, and are given a god/great/good/bad status based on these scores.

Each weapon supported by this website has had each of their perks rated by both FtpApoc and Mono Antumbra. You can check out Monos stuff here:
Twitter Youtube

If you see a perk with no backround or a masterwork with a 0/0 score I've messed up, please message me on discord at DaPoliceman#7777 and ill get it fixed!

You can check out the code for this page, and the whole website, on my Github
